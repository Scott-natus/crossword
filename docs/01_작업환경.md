#### 3. 포트 구성 요약

- **9000 포트**: Apache (SSL) → Laravel 서비스 (9090)
- **8080 포트**: Spring Boot 게시판
- **8081 포트**: Spring Boot 퍼즐게임
- **9090 포트**: Laravel 게시판+퍼즐게임
