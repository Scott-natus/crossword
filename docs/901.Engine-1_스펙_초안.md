# 901. Engine-1 스펙 초안 (v1)

> **목적**: Engine-0이 모은 트렌드 메타를 K-CrossWord **일일 테마**(`100` §8.1)와 정합한다.  
> **최종 갱신**: 2026-04-15

## 1. 범위 (v1)

| 구분 | 내용 |
|:---|:---|
| **입력** | PostgreSQL `engine0_trends` (최소 `title`, 선택 `summary_features`, `category`) |
| **출력** | `engine1_trend_theme_map` — 트렌드 1건당 **테마 코드 1개** (`K-POP`, `K-DRAMA`, `K-MOVIE`, `K-CULTURE`, `Korean`) |
| **방식** | 키워드 가중치 규칙 (`map_trend_themes.py`). Gemini 재분류는 v2 후보. |
| **비범위** | 단어·`cat2` 선정, `theme_daily_puzzles` 직접 생성, 영상 파일 분석 |

## 2. 테마 코드

`100.퍼즐게임의 작동원리와 서비스 개요.md` §8.1과 동일.

## 3. DB

- DDL: `harness-factory/engine-1-architect/sql/001_engine1_trend_theme_map.sql`
- `engine0_trends.id`에 FK, 트렌드당 매핑 1건(UNIQUE).

## 4. 실행

### DDL (psql)

`.env`에 이미 `NATUS_DB_PASSWORD` 등이 있으므로 **비밀번호를 다시 칠 필요 없음**:

```bash
cd harness-factory/engine-1-architect
./with_db.sh -f sql/001_engine1_trend_theme_map.sql
```

(`with_db.sh`가 `crossword/.env`를 읽고 `DATABASE_URL`을 만든 뒤 `psql` 실행)

### 매핑 스크립트

```bash
cd harness-factory/engine-1-architect
../engine-0-scanner/venv/bin/python map_trend_themes.py --dry-run --limit 10
../engine-0-scanner/venv/bin/python map_trend_themes.py --limit 500
```

연결 정보는 `crossword/.env` (`db_env` 동일).
