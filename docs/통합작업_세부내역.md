# 퍼즐게임-게시판 통합작업 세부내역

## 📋 작업 개요

### 요청사항
- **목표**: 8080(게시판)과 8081(퍼즐게임) Spring Boot 애플리케이션을 단일 애플리케이션(8080)으로 통합
- **도메인**: `natus250601.viewdns.net`
- **라우팅 구조**:
  - 메인 화면: `/` (기존 퍼즐게임 메인과 동일)
  - 게시판: `/board`
  - 퍼즐게임: `/Korean/game`, `/K-pop`, `/K-movie`, `/K-Drama`, `/K-Culture`
  - 관리자: `/admin/` (컨텍스트 패스 제거)

### 기술 스택
- **Spring Boot**: 8080 포트 통합 서비스
- **PostgreSQL**: 기존 데이터베이스 유지
- **Apache**: 리버스 프록시 (80 → 8080)
- **Redis**: 세션 관리
- **Thymeleaf**: 템플릿 엔진

## 🐛 해결된 문제들

### 1. 컴파일 오류
- **문제**: Lombok 의존성 누락
- **해결**: `build.gradle`에 Lombok 의존성 추가
- **문제**: 엔티티 import 경로 오류
- **해결**: 패키지 경로 수정 (`com.example.crossword` → `com.example.board`)

### 2. 데이터베이스 스키마 불일치
- **문제**: JPA 엔티티의 `Long` 타입과 DB의 `integer` 타입 불일치
- **해결**: 엔티티 타입을 `Integer`로 수정

### 3. Spring Boot 애플리케이션 시작 실패
- **문제**: Gemini API 설정 누락으로 인한 서비스 시작 실패
- **해결**: Gemini API 의존 서비스들을 임시 비활성화

### 4. 인증 API 404 오류
- **문제**: `/api/auth/status` 엔드포인트 누락
- **해결**: `AuthController`에 인증 상태 확인 API 추가

### 5. Spring Security 인증 문제
- **문제**: `/` 경로가 인증이 필요하다고 판단
- **해결**: SecurityConfig에서 `/` 경로를 명시적으로 `permitAll()` 설정

#### 6.2 게임 통계 동기화 시스템 구현
**구현된 기능**:
- `UserRepository.updateUserGameStatistics()`: 네이티브 쿼리로 게임 통계 업데이트
- `UserManagementService.updateUserGameStatistics()`: 트랜잭션 기반 통계 업데이트

**SQL 쿼리**:
```sql
UPDATE users SET 
    games_played = subquery.games_played,
    total_score = subquery.total_score,
    wins = subquery.wins
FROM (
    SELECT 
        user_id,
        COUNT(*) as games_played,
        COALESCE(SUM(score), 0) as total_score,
        COUNT(CASE WHEN game_status = 'completed' THEN 1 END) as wins
    FROM puzzle_game_records 
    GROUP BY user_id
) as subquery
WHERE users.id = subquery.user_id
```

## 7. 5회 오답 후 정답 표시 기능 수정 (2025-10-28)

### 문제 상황
- 5회 오답 후 정답을 보여주는 기능에서 모든 정답칸이 콤마(,)로 표시되는 문제 발생

3. **데이터베이스 재조회**: 게임 중에 데이터베이스의 단어가 변경되어 원본과 다른 단어가 표시됨

### 해결 과정
1. **8081 서비스 참조**: 정상 작동하는 8081 서비스의 코드를 분석
2. **데이터 구조 처리 개선**: `extracted_words`가 `Map` 또는 `List`인지 확인 후 적절히 처리
3. **올바른 필드 사용**: `pz_word_id`를 사용해서 데이터베이스에서 실제 단어 조회

```java
// extracted_words 구조 확인 (Map 또는 List)
if (extractedWordsObj instanceof Map) {
```
