# 서비스 환경 및 프레임워크, 아키텍처 설명

## 1. 개요
본 시스템은 대규모 서비스를 위한 다중 모듈 기반의 Java 웹 애플리케이션으로, 게시판 시스템(Board System)과 크로스워드 프로젝트(Crossword Project)가 독립적인 프로세스로 운영되는 구조입니다.

## 2. 기술 스택
- **언어**: Java 17+
- **프레임워크**: Spring Boot 3.x
- **빌드 도구**: Gradle (다중 모듈 및 독립 프로젝트 구성)
- **데이터베이스**: MySQL / MariaDB (JPA 기반 데이터 관리)
- **캐싱 및 서버**: Redis (Session 관리용), Apache/Nginx (Reverse Proxy)

## 3. 서비스 구성 (아키텍처)
시스템은 포트 기반으로 서비스가 분리되어 있습니다.

| 서비스명 | 포트 | 주요 기능 |
| :--- | :--- | :--- |
| **Board System (8080)** | 8080 | 게시판 관리, 사용자 커뮤니티 기능 제공 |
| **Crossword Project (8081)** | 8081 | 크로스워드 게임 로직, Gemini AI 기반 힌트 자동 생성 |

## 4. 핵심 모듈 설명
- **Gemini AI Integration**: `GeminiService`를 통해 Google Gemini API와 연동되어 크로스워드 단어에 대한 힌트를 자동으로 생성 및 저장합니다.
- **Scheduler**: `HintGenerationScheduler`가 주기적으로 힌트가 없는 단어를 확인하여 AI 서비스를 호출합니다.

## 5. 배포 환경
- **OS**: Linux (Ubuntu 기반)
- **웹 서버 서버 경로**: `/var/www/html`
- **프로젝트 소스 경로**:
    - `/var/www/html/java-projects` (8080 서비스)
    - `/var/www/html/crossword-projects` (8081 서비스)
